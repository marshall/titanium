Installing Gears

Visit gears.google.com to install Gears on your computer.

To find out how your web application can offer Gears installation to users, see Nuts and Bolts: Detecting and Installing Gears.

Uninstalling Gears

Windows XP or Vista

Uninstall Gears by using Add or Remove Programs (on XP) or Programs and Features (on Vista) from the Windows Control Panel.

  • In the Windows Control Panel, click Add or Remove Programs
  • Click on the entry for Gears
  • Click Remove

Mac OS/X and Linux

Uninstall Gears by removing the Gears Firefox extension.

  • In Firefox, select Tools > Extensions (Firefox 1.x) or Tools > Add-ons (Firefox 2.0+) from the menu
  • Click the Gears entry, and click Uninstall
  • Restart Firefox

Windows Mobile 5.0 and 6.0

Uninstall Gears by removing the Gears program.

  • Select Settings then, if present, select the System tab
  • Select Remove Programs
  • Click the Gears entry, and click Remove
  • Restart your device
